# Welcome to the RootKit HTML + CSS Certification Roadmap

## In this roadmap we will be laying out everything we will be testing for in our HTML + CSS Certification Test

### On the HTML Side we will be testing for the following.

1. Elements
2. Attributes
3. Headings
4. Paragraphs
5. Styles
6. Formatting
7. Quotations
8. Comments
9. Links
10. Images
11. Favicon
12. Tables
13. Lists
14. Block & Inline
15. Classes
16. Id
17. Layout
18. Symbols
19. Emojis
20. Charset

### On the CSS side we will be testing for the following.

1. Syntax
2. Backgrounds
3. Borders
4. Margins
5. Padding
6. Height/Width
7. Box Model
8. Outline
9. Text
10. Fonts
11. Icons
12. Links
13. Lists
14. Tables
15. Display
16. Max-width
17. Position
18. Overflow
19. Float
20. Inline-block
21. Align
22. Pseudo-class
23. Pseudo-element
24. Opacity
25. Navigation Bar
26. Dropdowns
27. Image Gallery
28. Image Sprites
29. Attr Selectors
30. Forms
31. Website Layout
